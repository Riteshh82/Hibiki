## 🛠️ Installation
1. **Clone the Repository**: `git clone https://github.com/Riteshh82/Hibiki.git`
2. **Install Dependencies**: `npm install`
3. **Set Up MongoDB**: Ensure MongoDB is running.
4. **Start the Development Server**: `npm run dev`
